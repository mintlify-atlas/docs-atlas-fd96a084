## Your First Script

Let's create a simple script that demonstrates the core capabilities of Minescript Plus. This script will display player information and respond to an in-game event.

<Steps>
  <Step title="Create a New Script">
    In your `/minescript` folder, create a new file called `my_first_script.py`.
  </Step>
  
  <Step title="Import Minescript Plus">
    Start by importing the classes you need:
    
    ```python my_first_script.py
    from minescript_plus import Player, World, Gui
    
    # Get player and world instances
    player = Player()
    world = World()
    
    # Display player information
    print("=== Player Info ===")
    print(f"Position: {player.blockPosition().toShortString()}")
    print(f"Game Mode: {Player.get_game_mode()}")
    print(f"XP Level: {Player.get_xp_levels()}")
    print(f"Food Level: {Player.get_food_level()}")
    print(f"Health: {player.getHealth()}")
    
    # Display world information
    print("\n=== World Info ===")
    print(f"Time of Day: {World.get_day_time()} ticks")
    print(f"Difficulty: {World.get_difficulty()}")
    print(f"Raining: {World.is_raining()}")
    print(f"Hardcore: {World.is_hardcore()}")
    
    # Show a welcome message
    Gui.set_title("Welcome!")
    Gui.set_subtitle("Minescript Plus is working")
    ```
  </Step>
  
  <Step title="Run Your Script">
    In Minecraft, open the chat and run:
    ```
    \my_first_script
    ```
    
    You should see your player and world information printed in the console, plus a title/subtitle appear on screen!
  </Step>
</Steps>

<Note>
You don't need to import all classes - just import the ones you need for your script.
</Note>

## Working with Inventory

Let's create a more practical example that interacts with your inventory:

```python inventory_example.py
from minescript import press_key_bind
from minescript_plus import Inventory, Screen
from time import sleep

# Open the inventory
press_key_bind("key.inventory", True)
if Screen.wait_screen():
    press_key_bind("key.inventory", False)
    
    # Swap items between inventory slot 22 and hotbar slot 4
    Inventory.inventory_hotbar_swap(22, 4)
    sleep(0.1)
    
    # Close the inventory
    Screen.close_screen()
    print("Item swap complete!")
else:
    print("Could not open inventory")
```

<Warning>
**Slot IDs Reference:**

- Player inventory hotbar: 0-8
- Player inventory main: 9-35
- Player inventory offhand: 40
- Player armor (boots, leggings, chestplate, helmet): 36-39
</Warning>

## Using Events

Minescript Plus includes a powerful event system. Here's an example that listens for actionbar changes:

```python event_example.py
import asyncio
from minescript import tick_loop
from minescript_plus import Event

loop = True

def on_actionbar(text: str):
    global loop
    print(f"Actionbar changed! Current value: {text}")
    loop = False

async def main():
    # Register the event listener
    listener = await Event.register("on_actionbar", on_actionbar)
    
    print("Waiting for actionbar change...")
    print("Test with: /title @s actionbar \"Hello World\"")
    
    # Wait for the event to trigger
    with tick_loop:
        while loop:
            await asyncio.sleep(0.05)
    
    # Clean up
    listener.unregister()
    print("Event listener stopped")

if __name__ == "__main__":
    asyncio.run(main())
```

<CodeGroup>
```python Decorator Style
from minescript_plus import Event

@Event.event
def on_actionbar(text: str):
    print(f"Actionbar: {text}")

# Later, in your main async function:
await Event.activate_all()
```

```python Manual Registration
from minescript_plus import Event

def on_actionbar(text: str):
    print(f"Actionbar: {text}")

# In your main async function:
listener = await Event.register("on_actionbar", on_actionbar)
```
</CodeGroup>

### Available Events

| Event Name | Triggers When | Returns |
|------------|---------------|----------|
| `on_title` | Title text changes | New title text |
| `on_subtitle` | Subtitle text changes | New subtitle text |
| `on_actionbar` | Actionbar text changes | New actionbar text |
| `on_open_screen` | Any screen opens | Screen name |

## Simulating Key Presses

You can simulate key presses programmatically:

```python key_example.py
from time import sleep
from minescript_plus import Key

# Wait a moment
sleep(1)

# Open inventory by pressing 'E'
Key.press_key("key.keyboard.e", True)
sleep(0.1)
Key.press_key("key.keyboard.e", False)
```

<Note>
See the [Minecraft Key Codes](https://minecraft.wiki/w/Key_codes#Current) reference for all available key names.
</Note>

## Advanced Example: Villager Trading

Here's a complete example that automates villager trading:

```python trading_example.py
from minescript_plus import Trading, Inventory, Screen
from time import sleep

# First, manually right-click a villager to open the trading screen
# Then run this script

# Get all available offers
offers = Trading.get_offers()

if offers is None:
    print("No trading screen open!")
else:
    print(f"Found {offers.size()} offers\n")
    
    # Display all trades
    for i in range(offers.size()):
        costA = Trading.get_costA(i, name_and_count=True)
        costB = Trading.get_costB(i, name_and_count=True)
        result = Trading.get_result(i, name_and_count=True)
        
        print(f"Offer {i}:")
        print(f"  Cost A: {costA}")
        if costB:
            print(f"  Cost B: {costB}")
        print(f"  Result: {result}")
        print()
    
    # Trade the first offer
    print("Trading offer 0...")
    Trading.trade_offer(0)
    sleep(0.5)
    
    Screen.close_screen()
    print("Trade complete!")
```

## Getting User Input

Minescript Plus provides a replacement for Python's `input()` that works in Minecraft:

```python input_example.py
from minescript_plus import input

name = input("What's your name? ")
print(f"Hello, {name}!")

# Save to chat history (optional)
favorite_color = input("Favorite color? ", save_history=True)
print(f"Nice! {favorite_color} is a great color.")
```
