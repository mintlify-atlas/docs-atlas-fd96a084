The `Player` class provides methods for retrieving player information and state.

<Note>
  The Player class can be instantiated to get direct access to the player instance: `player = Player()`
</Note>

## get_latency

Retrieves the player's network latency (ping).

**Returns:** `int` - The latency in milliseconds.

**Source:** `minescript_plus.py:878`

```python
import minescript_plus as ms

latency = ms.Player.get_latency()
print(f"Ping: {latency}ms")
```

## get_player_block_position

Returns the player's current block position as a list of integers.

**Returns:** `list[int]` - The [x, y, z] coordinates of the player's block position.

**Source:** `minescript_plus.py:949`

```python
import minescript_plus as ms

pos = ms.Player.get_player_block_position()
print(f"Player is at block: {pos[0]}, {pos[1]}, {pos[2]}")
```

<Tip>
  This method uses `floor()` to convert the player's exact position to block coordinates.
</Tip>

## Complete Example: Player Status Display

```python
import minescript_plus as ms

def display_player_status():
    """Display comprehensive player status."""
    
    # Game mode
    mode = ms.Player.get_game_mode()
    
    # Position
    pos = ms.Player.get_player_block_position()
    
    # Food and saturation
    food = ms.Player.get_food_level()
    saturation = ms.Player.get_saturation_level()
    
    # XP
    levels = ms.Player.get_xp_levels()
    progress = ms.Player.get_experience_progress()
    
    # Network
    latency = ms.Player.get_latency()
    
    print("=== Player Status ===")
    print(f"Game Mode: {mode}")
    print(f"Position: {pos[0]}, {pos[1]}, {pos[2]}")
    print(f"Food: {food}/20 (Saturation: {saturation:.1f})")
    print(f"XP: Level {levels} ({progress * 100:.1f}% to next)")
    print(f"Latency: {latency}ms")

display_player_status()
```

## Health and Hunger Monitor

```python
import minescript_plus as ms
import time

def monitor_vitals(duration: int = 60):
    """Monitor player vitals for a duration."""
    start_time = time.time()
    
    while time.time() - start_time < duration:
        food = ms.Player.get_food_level()
        
        # Get health from player instance
        player = ms.Player()
        health = player.getHealth()
        
        # Warn if low
        if health < 6:
            print("⚠️ WARNING: Low health!")
        if food < 6:
            print("⚠️ WARNING: Low hunger!")
        
        time.sleep(5)

monitor_vitals(60)
```
