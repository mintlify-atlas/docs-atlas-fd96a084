The `Util` class provides various utility functions for common tasks in Minecraft scripting.

## Job Management

### get_job_id

Returns the job_id of a job matching the given command string.

<ParamField path="cmd" type="str" required>
  The command string to search for among running jobs.
</ParamField>

**Returns:** `int | None` - The job_id of the matching job, or None if not found.

**Source:** `minescript_plus.py:1437`

```python
import minescript_plus as ms

job_id = ms.Util.get_job_id("my_script.py")
if job_id:
    print(f"Script is running with job ID: {job_id}")
else:
    print("Script is not running")
```

### set_clipboard

Sets the system clipboard to the specified string.

<ParamField path="string" type="str" required>
  The text to copy to the clipboard.
</ParamField>

**Returns:** `None`

**Source:** `minescript_plus.py:1458`

```python
import minescript_plus as ms
import minescript as msc

# Copy player position to clipboard
pos = msc.player_position()
coords = f"{pos[0]:.2f}, {pos[1]:.2f}, {pos[2]:.2f}"
ms.Util.set_clipboard(coords)
print(f"Copied to clipboard: {coords}")
```

## Distance Calculation

### get_distance

Calculates the Euclidean distance between two 3D positions.

<ParamField path="pos1" type="list" required>
  The first position as [x, y, z].
</ParamField>

<ParamField path="pos2" type="list" default="None">
  The second position as [x, y, z]. If None, uses current player position.
</ParamField>

**Returns:** `float` - The Euclidean distance between pos1 and pos2.

**Source:** `minescript_plus.py:1468`

```python
import minescript_plus as ms
import minescript as msc

# Distance from player to a location
spawn = [0, 64, 0]
distance = ms.Util.get_distance(spawn)
print(f"Distance to spawn: {distance:.2f} blocks")

# Distance between two specific points
pos1 = [100, 64, 100]
pos2 = [200, 64, 200]
distance = ms.Util.get_distance(pos1, pos2)
print(f"Distance: {distance:.2f} blocks")
```

## NBT Handling

### get_nbt

Get a value from nested SNBT data using dot notation.

<ParamField path="obj" type="dict" required>
  Parsed SNBT object (dictionary).
</ParamField>

<ParamField path="path" type="str" required>
  Dot-separated path (e.g., "Inventory.0.id").
</ParamField>

<ParamField path="default" type="any" default="None">
  Default value if path not found.
</ParamField>

**Returns:** `Any` - Value at the specified path or default.

**Source:** `minescript_plus.py:1488`

```python
import minescript_plus as ms
import lib_nbt

# Parse SNBT
snbt = '{Inventory:[{id:"minecraft:diamond",Count:64b}]}'
nbt = lib_nbt.parse_snbt(snbt)

# Get nested value
item_id = ms.Util.get_nbt(nbt, "Inventory.0.id")
count = ms.Util.get_nbt(nbt, "Inventory.0.Count", default=0)

print(f"Item: {item_id}, Count: {count}")
```

## Light Level

### get_light_level

Returns the light level at a specified block position.

<ParamField path="block_pos" type="BlockPos" default="None">
  The position to check. If None, uses player's current block position.
</ParamField>

<ParamField path="source" type="str" default="'RAW'">
  Type of light: "RAW" (total brightness), "SKY" (skylight), or "BLOCK" (block light).
</ParamField>

**Returns:** `int` - The light level (0-15).

**Source:** `minescript_plus.py:1527`

```python
import minescript_plus as ms

# Get light level at player position
light = ms.Util.get_light_level()
print(f"Light level: {light}")

# Get skylight only
sky_light = ms.Util.get_light_level(source="SKY")
print(f"Sky light: {sky_light}")

# Get block light only
block_light = ms.Util.get_light_level(source="BLOCK")
print(f"Block light: {block_light}")

# Check if safe from mobs (light >= 8)
if light >= 8:
    print("Safe from mob spawns")
else:
    print("Mobs can spawn here!")
```

## Sound

### play_sound

Plays a sound on the client side.

<ParamField path="sound" type="SoundEvent" default="None">
  The sound event to play. Defaults to EXPERIENCE_ORB_PICKUP if None.
</ParamField>

<ParamField path="sound_source" type="SoundSource" default="None">
  The source of the sound. Defaults to PLAYERS if None.
</ParamField>

<ParamField path="volume" type="float" default="1.0">
  Volume (0.0 to 1.0).
</ParamField>

<ParamField path="pitch" type="float" default="1.0">
  Pitch (0.0 to 2.0).
</ParamField>

**Returns:** `None`

**Source:** `minescript_plus.py:1549`

```python
import minescript_plus as ms

# Play default sound (XP pickup)
ms.Util.play_sound()

# Play custom sound
SoundEvents = ms.Util.get_soundevents()
SoundSource = ms.Util.get_soundsource()

ms.Util.play_sound(
    sound=SoundEvents.BLOCK_NOTE_BLOCK_PLING,
    sound_source=SoundSource.BLOCKS,
    volume=1.0,
    pitch=2.0
)
```

## Complete Examples

### Light Monitor

```python
import minescript_plus as ms
import time

def monitor_light():
    """Monitor light levels and warn if too dark."""
    
    print("Monitoring light levels... (Ctrl+C to stop)")
    
    while True:
        total = ms.Util.get_light_level(source="RAW")
        sky = ms.Util.get_light_level(source="SKY")
        block = ms.Util.get_light_level(source="BLOCK")
        
        print(f"Light - Total: {total}, Sky: {sky}, Block: {block}")
        
        if total < 8:
            print("⚠️ WARNING: Mobs can spawn here!")
            # Play warning sound
            SoundEvents = ms.Util.get_soundevents()
            ms.Util.play_sound(
                sound=SoundEvents.BLOCK_NOTE_BLOCK_BASS,
                volume=0.5,
                pitch=0.5
            )
        
        time.sleep(2)

monitor_light()
```

### Coordinate Copier

```python
import minescript_plus as ms
import minescript as msc
from minescript import render_loop

def copy_coords_to_clipboard():
    """Copy current coordinates to clipboard and show toast."""
    
    # Get position
    pos = msc.player_position()
    coords = f"{pos[0]:.0f}, {pos[1]:.0f}, {pos[2]:.0f}"
    
    # Copy to clipboard
    ms.Util.set_clipboard(coords)
    
    # Show notification
    with render_loop:
        ms.Util.show_toast(
            "Coordinates Copied",
            coords
        )
    
    print(f"Copied to clipboard: {coords}")

copy_coords_to_clipboard()
```

### Distance Tracker

```python
import minescript_plus as ms
import minescript as msc
import time

def track_distance_from_point(target_pos: list):
    """Track distance from a specific point."""
    
    print(f"Tracking distance from {target_pos}")
    print("Press Ctrl+C to stop")
    
    while True:
        distance = ms.Util.get_distance(target_pos)
        print(f"Distance: {distance:.2f} blocks", end='\r')
        
        # Alert if getting close
        if distance < 10:
            SoundEvents = ms.Util.get_soundevents()
            ms.Util.play_sound(
                sound=SoundEvents.BLOCK_NOTE_BLOCK_PLING,
                pitch=2.0
            )
            print("\n⚠️ Getting close!")
        
        time.sleep(0.5)

# Track distance from spawn
track_distance_from_point([0, 64, 0])
```
