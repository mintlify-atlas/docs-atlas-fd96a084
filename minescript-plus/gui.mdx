The `Gui` class provides methods for getting and setting title, subtitle, and actionbar text in Minecraft.

<Note>
  All methods in the Gui class must be called within the `render_loop` context.
</Note>

## Getting Text

### get_title

Retrieves the current title text displayed on screen.

**Returns:** `str | None` - The title text, or None if not available.

**Source:** `minescript_plus.py:642`

```python
import minescript_plus as ms
from minescript import render_loop

with render_loop:
    title = ms.Gui.get_title()
    if title:
        print(f"Current title: {title}")
```

### get_actionbar

Retrieves and clears the current actionbar (overlay message) text.

**Returns:** `str | None` - The current overlay message string if present, otherwise None.

**Source:** `minescript_plus.py:668`

<Warning>
  This method clears the actionbar after reading it.
</Warning>

```python
import minescript_plus as ms
from minescript import render_loop

with render_loop:
    actionbar = ms.Gui.get_actionbar()
    if actionbar:
        print(f"Actionbar message: {actionbar}")
```

## Title Timing

### set_title_times

Sets the timing for the title and subtitle display.

<ParamField path="fadeInTicks" type="int" required>
  Number of ticks for the title to fade in.
</ParamField>

<ParamField path="stayTicks" type="int" required>
  Number of ticks for the title to stay visible.
</ParamField>

<ParamField path="fadeOutTicks" type="int" required>
  Number of ticks for the title to fade out.
</ParamField>

**Returns:** `None`

**Source:** `minescript_plus.py:721`

```python
import minescript_plus as ms
from minescript import render_loop

with render_loop:
    # Fade in 10 ticks, stay 70 ticks, fade out 20 ticks
    ms.Gui.set_title_times(10, 70, 20)
    ms.Gui.set_title("Custom Timing")
```

### reset_title_times

Resets the title and subtitle display times to the default values.

**Returns:** `None`

**Source:** `minescript_plus.py:736`

## Complete Example

```python
import minescript_plus as ms
from minescript import render_loop
import time

with render_loop:
    # Set custom timing
    ms.Gui.set_title_times(10, 100, 20)
    
    # Display title and subtitle
    ms.Gui.set_title("Quest Started")
    ms.Gui.set_subtitle("Find the hidden treasure")
    
    time.sleep(2)
    
    # Display actionbar message
    ms.Gui.set_actionbar("Progress: 0/10")
    
    time.sleep(3)
    
    # Clear everything
    ms.Gui.clear_titles()
    ms.Gui.reset_title_times()
```
