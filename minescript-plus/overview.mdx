## About Minescript Plus

Minescript Plus is a user-friendly Python API that extends the [Minescript mod](https://github.com/maxuser0/minescript) with additional functionality. It provides high-level interfaces for interacting with Minecraft's inventory system, GUI elements, world data, player information, and more.

<Note>
  **Version**: 0.16.2-alpha

## Architecture

Minescript Plus is built on top of Minescript's Java interop layer, providing:

- **High-level Python APIs** - Clean, Pythonic interfaces
- **Direct Java access** - Some classes expose underlying Minecraft instances
- **Type hints** - Full type annotations for IDE support
- **Async support** - Event system uses asyncio

### Direct Minecraft Class Access

Some classes (`Client`, `Player`, `World`) expose the original Minecraft class instance when initialized:

```python
from minescript_plus import Player

p = Player()
print(p.getArmorValue())  # Call any public method from mc.player
```

## Module Structure

The API is organized into logical modules:

| Module | Purpose |
|--------|----------|
| `Inventory` | Container and item manipulation |
| `Screen` | Screen detection and control |
| `Gui` | Title, subtitle, actionbar management |
| `Key` | Key press simulation |
| `Client` | Client-level operations |
| `Player` | Player state and information |
| `Server` | Server info and tablist |
| `World` | World data and block interaction |
| `Trading` | Villager trading automation |
| `Hud` | HUD text and item rendering (Fabric API required) |
| `Util` | Utility functions |
| `Event` | Event system for GUI changes |
| `Keybind` | Custom keybind registration |

## Usage Pattern

Typical import and usage:

```python
from minescript_plus import Inventory, Gui, Player, Event

# Use static methods directly
Gui.set_title("Hello!")
pos = Player.get_player_block_position()

# Open inventory and click slot
Inventory.open_targeted_chest()
Inventory.click_slot(0)

# Register event listener
async def on_title(text: str):
    print(f"Title: {text}")

await Event.register("on_title", on_title)
```

## Requirements

<Warning>
  Minescript Plus requires specific versions and dependencies:
</Warning>

- **Minecraft**: 1.21.5 - 1.21.11
- **Minescript**: 5.0b6 or newer
- **Python**: 3.10 or higher
- **Mappings**: Required for Fabric (run `\install_mappings` in chat)
- **Fabric API mod**: Optional, required for `Hud` class only
- **lib_nbt v1**: Optional, required for `Inventory.find_item()` only
