A comprehensive enumeration of GLFW keyboard key codes with their numeric values, compatible with GLFW applications.

## Overview

The `glfw_keycodes.py` module provides:

- **GLFWKey enum**: IntEnum with all standard GLFW key codes
- **get_glfw_keycode()**: Function to look up key codes by name
- Full compatibility with official GLFW C/C++ key constants

## GLFWKey Enum

The `GLFWKey` enum inherits from `IntEnum`, so values can be used directly as integers in comparisons and arithmetic operations.

## Functions

### get_glfw_keycode()

Look up a GLFW key code by name (case-insensitive).

```python
get_glfw_keycode(key_name: str) -> int
```

**Parameters:**
- `key_name` (str): Name of the key (case-insensitive)

**Returns:**
- `int`: The key code value, or `GLFWKey.UNKNOWN` (-1) if not found

## Usage

<Steps>

### Import the Module

```python
from glfw_keycodes import GLFWKey, get_glfw_keycode
```

### Use Enum Values

Access key codes directly from the enum:

```python
space_key = GLFWKey.SPACE         # 32
escape_key = GLFWKey.ESCAPE       # 256
a_key = GLFWKey.A                 # 65
f1_key = GLFWKey.F1               # 290
enter_key = GLFWKey.ENTER         # 257
```

### Use the Lookup Function

Get key codes by name (case-insensitive):

```python
w_key = get_glfw_keycode('W')           # 87
w_key = get_glfw_keycode('w')           # 87 (case-insensitive)
enter_key = get_glfw_keycode('enter')   # 257
space_key = get_glfw_keycode('SPACE')   # 32

# Unknown keys return -1
unknown = get_glfw_keycode('invalid')   # -1
```

### Check Key Membership

Verify if a key exists in the enum:

```python
if GLFWKey.LEFT_SHIFT in GLFWKey:
    print(f"Left Shift: {GLFWKey.LEFT_SHIFT}")  # 340
```

### Use in Comparisons

Since `GLFWKey` is an `IntEnum`, values work as integers:

```python
key_code = 65

if key_code == GLFWKey.A:
    print("A key pressed")

# Arithmetic operations
if key_code >= GLFWKey.A and key_code <= GLFWKey.Z:
    print("Letter key pressed")
```

### Iterate Over Keys

Loop through all available keys:

```python
for key in GLFWKey:
    print(f"{key.name}: {key.value}")
```

</Steps>

## Complete Example

```python
from glfw_keycodes import GLFWKey, get_glfw_keycode

# Direct enum access
print(f"SPACE key: {GLFWKey.SPACE}")         # 32
print(f"ESC key: {GLFWKey.ESCAPE}")          # 256
print(f"A key: {GLFWKey.A}")                 # 65
print(f"F1 key: {GLFWKey.F1}")               # 290

# Function-based lookup
print(f"W key: {get_glfw_keycode('W')}")     # 87
print(f"Enter: {get_glfw_keycode('enter')}") # 257

# Check key membership
if GLFWKey.LEFT_SHIFT in GLFWKey:
    print(f"Left Shift: {GLFWKey.LEFT_SHIFT}")  # 340

# Use in key handler function
def handle_key_press(key_code):
    if key_code == GLFWKey.ESCAPE:
        print("Escape pressed - exiting")
    elif key_code == GLFWKey.SPACE:
        print("Space pressed - jumping")
    elif key_code >= GLFWKey.A and key_code <= GLFWKey.Z:
        letter = chr(key_code)
        print(f"Letter pressed: {letter}")
    elif key_code >= GLFWKey.F1 and key_code <= GLFWKey.F12:
        fn_num = key_code - GLFWKey.F1 + 1
        print(f"F{fn_num} pressed")

# Test the handler
handle_key_press(GLFWKey.SPACE)    # "Space pressed - jumping"
handle_key_press(GLFWKey.A)        # "Letter pressed: A"
handle_key_press(GLFWKey.F5)       # "F5 pressed"
```

## Integration with GLFW

Use with the GLFW library for keyboard input:

```python
import glfw
from glfw_keycodes import GLFWKey

def key_callback(window, key, scancode, action, mods):
    if action == glfw.PRESS:
        if key == GLFWKey.ESCAPE:
            glfw.set_window_should_close(window, True)
        elif key == GLFWKey.SPACE:
            print("Jump!")
        elif key == GLFWKey.W:
            print("Move forward")
        elif key == GLFWKey.S:
            print("Move backward")

# Set up GLFW window and callback
glfw.init()
window = glfw.create_window(640, 480, "Game", None, None)
glfw.set_key_callback(window, key_callback)
```
