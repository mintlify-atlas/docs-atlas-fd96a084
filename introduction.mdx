## What is Minescript Plus?

Minescript Plus is a feature-rich API extension for the [Minescript](https://github.com/maxuser0/minescript) mod that adds extra functionality and provides a user-friendly interface for creating powerful Minecraft scripts in Python.

**Version:** 0.16.2-alpha  
**Author:** RazrCraft

<Note>
Minescript Plus is actively developed and should be imported by other scripts, not run directly.
</Note>

## Direct Minecraft Class Access

Some classes in Minescript Plus—specifically **Client**, **Player**, and **World**—expose the original Minecraft class instance when initialized. This means you can access their public fields and methods directly:

```python
from minescript_plus import Player

p = Player()
print(p.getArmorValue())  # Access any public method from Minecraft's Player class
```
