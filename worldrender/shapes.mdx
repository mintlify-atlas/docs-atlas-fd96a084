## Boxes

Render a wireframe cuboid between two corner points.

### add_box

```python
WorldRender.add_box(x1, y1, z1, x2, y2, z2, r=255, g=255, b=255, a=255, always_on_top=True)
```

<ParamField path="x1" type="int" required>
  X coordinate of first corner (integer grid position)
</ParamField>

<ParamField path="y1" type="int" required>
  Y coordinate of first corner (integer grid position)
</ParamField>

<ParamField path="z1" type="int" required>
  Z coordinate of first corner (integer grid position)
</ParamField>

<ParamField path="x2" type="int" required>
  X coordinate of second corner (integer grid position)
</ParamField>

<ParamField path="y2" type="int" required>
  Y coordinate of second corner (integer grid position)
</ParamField>

<ParamField path="z2" type="int" required>
  Z coordinate of second corner (integer grid position)
</ParamField>

<ParamField path="r" type="int" default="255">
  Red channel (0-255)
</ParamField>

<ParamField path="g" type="int" default="255">
  Green channel (0-255)
</ParamField>

<ParamField path="b" type="int" default="255">
  Blue channel (0-255)
</ParamField>

<ParamField path="a" type="int" default="255">
  Alpha/opacity (0-255)
</ParamField>

<ParamField path="always_on_top" type="bool" default="true">
  If True, renders through blocks
</ParamField>

**Returns:** `int` - Unique ID for this box

### remove_box

```python
WorldRender.remove_box(x1=None, y1=None, z1=None, x2=None, y2=None, z2=None, id=None)
```

Remove a box by coordinates or by ID.

### get_box_list

```python
WorldRender.get_box_list()
```

**Returns:** `dict` - Mapping of IDs to `(x1, y1, z1, x2, y2, z2, r, g, b, a, always_on_top)` tuples

## Rectangles

Render a quadrilateral defined by four corner vertices.

### add_rect

```python
WorldRender.add_rect(
    x1, y1, z1,
    x2, y2, z2,
    x3, y3, z3,
    x4, y4, z4,
    r=255, g=255, b=255, a=255,
    filled=False, always_on_top=True
)
```

<ParamField path="x1, y1, z1" type="float" required>
  First corner position
</ParamField>

<ParamField path="x2, y2, z2" type="float" required>
  Second corner position
</ParamField>

<ParamField path="x3, y3, z3" type="float" required>
  Third corner position
</ParamField>

<ParamField path="x4, y4, z4" type="float" required>
  Fourth corner position
</ParamField>

<ParamField path="r" type="int" default="255">
  Red channel (0-255)
</ParamField>

<ParamField path="g" type="int" default="255">
  Green channel (0-255)
</ParamField>

<ParamField path="b" type="int" default="255">
  Blue channel (0-255)
</ParamField>

<ParamField path="a" type="int" default="255">
  Alpha/opacity (0-255)
</ParamField>

<ParamField path="filled" type="bool" default="false">
  If True, renders a filled quad; otherwise renders an outline
</ParamField>

<ParamField path="always_on_top" type="bool" default="true">
  If True, renders through blocks
</ParamField>

**Returns:** `int` - Unique ID for this rectangle

<Note>
Vertices should be specified in order (clockwise or counter-clockwise) to form a valid planar quad.
</Note>

### remove_rect

```python
WorldRender.remove_rect(
    x1=None, y1=None, z1=None,
    x2=None, y2=None, z2=None,
    x3=None, y3=None, z3=None,
    x4=None, y4=None, z4=None,
    id=None
)
```

Remove a rectangle by four corner coordinates or by ID.

### get_rect_list

```python
WorldRender.get_rect_list()
```

**Returns:** `dict` - Mapping of IDs to `(x1, y1, z1, x2, y2, z2, x3, y3, z3, x4, y4, z4, r, g, b, a, filled, always_on_top)` tuples
