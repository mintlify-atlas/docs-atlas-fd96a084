## What is Minescript Scripts?

Minescript Scripts is a comprehensive collection of Python tools for enhancing your Minecraft experience with the Minescript mod. The project features **Minescript Plus**, a powerful API extension, along with world rendering utilities, configuration helpers, and example scripts.

## Quick Start

Get up and running with Minescript Plus in minutes.

<Steps>
  <Step title="Install Minescript">
    Install the Minescript mod for Minecraft 1.21.5 - 1.21.11 with Python 3.10+
  </Step>
  <Step title="Download Minescript Plus">
    Download `minescript_plus.py` and place it in your `/minescript` folder
  </Step>
  <Step title="Install Mappings">
    Run `\install_mappings` in Minecraft chat (required for Fabric)
  </Step>
  <Step title="Import and Use">
    Import the module in your scripts and start building
  </Step>
</Steps>

<Card title="View Quickstart Guide" icon="rocket" href="/quickstart">
  Follow the complete setup guide with code examples
</Card>

## Requirements

- **Minecraft**: 1.21.5 - 1.21.11
- **Minescript**: 5.0b6 or newer
- **Python**: 3.10 or higher
- **Mappings**: Required for Fabric (run `\install_mappings` in chat)
- **Fabric API mod**: Optional, required for HUD class only
- **lib_nbt v1**: Optional, required for `Inventory.find_item()` only
